# EstateVisio SEO Fixes Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Fix all high and medium SEO issues found in the site audit: missing OG images on inner pages, weak H1s, non-locale-prefixed internal links, image loading attributes, and page-specific schema markup.

**Architecture:** All metadata lives in `app/[lang]/<page>/page.tsx` via Next.js `generateMetadata`. H1 copy lives in `config/copy.ts`. Components render content from that copy config. Schema is injected as JSON-LD `<script>` tags in each page.tsx, following the same pattern already used in `app/[lang]/layout.tsx`.

**Tech Stack:** Next.js (App Router, `[lang]` dynamic segment), TypeScript, React. No external SEO libraries. Read `node_modules/next/dist/docs/` if unsure about any Next.js API, as this version may differ from training data.

## Task 6: Fix Non-Locale-Prefixed Internal Links in Home Component

**Files:**
- Modify: `components/pages/Home.tsx`

**The problem:** At the top of `Home.tsx`, there's a module-level `SERVICES` constant (defined outside the component) with hardcoded non-locale-prefixed hrefs:

```ts
const SERVICES = [
  { id: '01', image: '/images/gallery-tour-bg.jpg', href: '/gallery#scene-01', copy: 'service1' as const },
  { id: '02', image: '/images/about-architecture.jpg', href: '/gallery#scene-02', copy: 'service2' as const },
];
```

These link to `/gallery#scene-01` instead of `/${lang}/gallery#scene-01`. The `lang` variable is only available inside the component from `useParams()`. The fix is to compute the `href` dynamically inside the component using the already-available `lang` variable.

```ts
];
```
